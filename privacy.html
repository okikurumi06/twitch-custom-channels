プライバシーポリシー

Twitch カスタムチャンネルリスト（以下「本拡張機能」）は、ユーザーの Twitch サイドバーに指定チャンネルを表示する機能のみを提供します。本拡張機能は、ユーザーのプライバシーを尊重し、以下の方針に従って運用されます。

1. 取得する情報

  • 本拡張機能は、個人を特定できる情報（氏名、メールアドレス、アカウント情報など）を一切収集しません。
  • ユーザーが入力した Twitch チャンネル名は、ブラウザのローカルストレージ（chrome.storage.local）にのみ保存されます。

2. プロフィール画像取得について

チャンネルアイコンの表示のため、Twitch の公開 API に対してプロキシサーバー経由でチャンネル名を送信します。この通信には、ユーザーの個人情報は含まれません。

3. トラッキング・分析

本拡張機能は、Google Analytics や広告トラッカーなどの分析ツールを一切使用していません。

4. お問い合わせ

ご質問・ご不明点がある場合は、拡張機能の配布ページまたは GitHub リポジトリ上でご連絡ください。

Privacy Policy (English)

Twitch Custom Channel List is a Chrome extension that allows users to display selected channels in the Twitch sidebar.

  • No personal information is collected, stored, or transmitted.
  • Channel names entered by users are stored only in chrome.storage.local.
  • A public proxy is used to fetch Twitch profile images, transmitting only the channel name.
  • No analytics, ads, or tracking technologies are used.
